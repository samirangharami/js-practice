# Task Status Representation

Use the following conventions:

| Status   | Markdown Format       |
| -------- | --------------------- |
| Not Done | `- [ ] Task text`     |
| Done     | `- [x] ~~Task text~~` |

Example:

```
- [x] ~~Buy groceries~~
```
